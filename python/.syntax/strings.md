### replace
> string = replace(old, new[, max])
```
fruit = 'strawberry'
print(fruit.replace('r', R)) stRawbeRRy
```

### find
> index = find(sub[, start[, end]])
```
exStr = "Mondo"
print(exStr.find("o")) 1
print(exStr.find("o", 4)) 4
```
